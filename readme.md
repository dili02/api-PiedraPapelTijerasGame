# API (ExpressJS, MongoDB, Joi, Bcrypt, JWT)

## Variables de entorno:
Crear archivo `.env` con variables de entorno:
```
DB_HOST=tu_DB_HOST
DB_PORT=tu_DB_PORT
DB_NAME=tu_DatabaseName
JWT_KEY=tu_JWT_KEY
```

## Instalación y Ejecución:
```bash

### Instalar dependencias
npm install

### Ejecutar API
npm run dev
```
